# Project name

## Requirement
- PHP 8
- Composer
- MySQL

## Install dependencies
On the terminal run.
```bash
composer install
```

## Setup database
On `src/config/config.php` you can change `HOST`, `DATABASE`, `USER`, `PASSWORD` constants.

## Run database scripts
You can run SQL script on `database/migrations.sql`.

## Run project
On the terminal run.
```bash
composer start
```

## Run unit tests
On the terminal run.
```bash
composer test
```
